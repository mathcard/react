Inicianco projeto
yarn init -y

Iniciando servidor
node src/server.js

Express - Trabalha com as rotas
instalação: yarn add express

Nodemon - Reincia servidor automaticamente
instalação: yarn add nodemon -D

Mongose
instalação: yarn add mongoose

Axios
instalação: yarn add axios

Cors - Permitir acessar api externo, dever ser chamando antes da rota
Instalação: yarn add cors

yarn.lock - Cache, armazena as dependencias
